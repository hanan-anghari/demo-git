# Demo d'un projet git

## Présentation:
ce projet est géré par git 

## Commande git 
* Pour initialiser git dans un dossier :
``` 
git init

```

* voir les changements depuis la derniere sauvegarde:
```
git status

```
* ajouter les dernieres modifications du fichier
`README.md`:
```
git add README.md
```
